Clack.Middleware.Stdout - Clack Middleware to set standard-output.

SYNOPSIS

(clackup (builder  
          <clack-middleware-stdout>  
          (lambda (env)  
            (format nil "You've got an access!~%")  
            '(200 nil ("Hi!")))))

DESCRIPTION

Clack.Middleware.Stdout allows you to set *standard-output* in an each requests.
