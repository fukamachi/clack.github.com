Clack.Logger.File - Output log messages to a file.

SYNOPSIS

(clackup (builder  
          (<clack-middleware-logger>  
           :logger (make-instance '<clack-logger-file>  
                      :output-file #p"~/server.log"))  
          (lambda (env)  
            (log-message :notice "You've got an access!")  
            '(200 nil ("ok")))))
