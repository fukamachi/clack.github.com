Clack.App.Directory - Server static files from document root with directory index.

SYNOPSIS

;; mount /var/www/ to http://localhost:5000/  
(clackup  
 (make-instance '<clack-app-directory>  
    :root #p"/var/www/")  
 :port 5000)
